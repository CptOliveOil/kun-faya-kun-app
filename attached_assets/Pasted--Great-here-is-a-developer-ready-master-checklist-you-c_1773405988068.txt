
Great — here is a developer-ready master checklist you can follow when building the next version of your app. I structured it like a product roadmap so it’s easier to implement step-by-step.

PHASE 1 — Critical Fixes (Do First)

These are things that are currently broken or incorrect.

1. Fix YouTube video playback

Problem: Videos show “unavailable”.

Solution

Convert all links into YouTube embed format.

Store:

videoId
title
speaker
duration

Generate:

Embed URL

https://www.youtube.com/embed/VIDEO_ID

Thumbnail

https://img.youtube.com/vi/VIDEO_ID/hqdefault.jpg

Example data structure

{
 title: "The Power of Bismillah",
 speaker: "Mufti Menk",
 videoId: "abc123xyz"
}

Player

<iframe
 src="https://www.youtube.com/embed/VIDEO_ID"
 allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture"
 allowfullscreen>
</iframe>

Add fallback:

If embed fails → show button
Open in YouTube

⸻

2. Fix video thumbnails

Current thumbnails are placeholders.

Solution

Use YouTube thumbnail endpoint

https://img.youtube.com/vi/VIDEO_ID/hqdefault.jpg

Add fallback image:

onError -> show placeholder thumbnail


⸻

3. Fix Prayer Arc Animation

Current animation looks visually off.

Problems

• Prayer markers not aligned properly
• Sun position not accurate
• Motion slightly awkward

Fix

Calculate sun position using current time between sunrise and sunset.

Formula concept:

progress = (currentTime - sunrise) / (sunset - sunrise)

Position the sun along the arc using that value.

Also:

• Highlight current prayer
• Fade past prayers
• Emphasize next prayer

⸻

4. Fix prayer notification bell icon

Bell icon doesn’t fit container.

Fix

Standardize icon button size.

Use:

button size: 44px x 44px
icon size: 20px
padding: 12px

Center icon vertically and horizontally.
